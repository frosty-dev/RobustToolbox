LoadAndStartReplayAsync(IWritableDirProvider, ResPath, LoadReplayCallback)

Convenience function that combines LoadReplayAsync(IWritableDirProvider, ResPath, LoadReplayCallback) and StartReplayAsync(ReplayData, LoadReplayCallback)

Declaration
Task<ReplayData> LoadAndStartReplayAsync(IWritableDirProvider dir, ResPath path, LoadReplayCallback callback = null)
Parameters
Type Name Description
IWritableDirProvider dir

A directory containing the replay files.
ResPath path

The path to the replay's subdirectory.
LoadReplayCallback callback

A callback delegate that invoked to provide information about the current loading progress. This callback can be used to invoke SuspendIfOutOfTime().
Returns
Type Description
Task<ReplayData>
Remarks

This task is intended to be used with a Job<T> so that the loading can happen over several frame updates. Note that a load is being processed over multiple "ticks", then the normal system tick updating needs to be blocked by subscribing to TickUpdateOverride in order to avoid errors while systems iterate over pre-init or pre-startup entities.
